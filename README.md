# Seatplanner

## Overview

Seatplanner is a Java application for managing and generating seat plans. It leverages MySQL for database operations and Apache POI for handling Excel files.

## Prerequisites

- **Java JDK 11 or higher**
- **MySQL Connector/J**
- **Apache POI**
